## 📰 重要博客文章
### Claude Code 现在支持工件功能
Claude AI 官方博客，Anthropic 公司
Claude Code 现在可以将工作进展捕捉为工件，这使得 Claude Code 的工作转化为可实时共享的视觉页面，包括 PR 演示、系统解释、仪表板和发布检查列表，且在会话进行时会自动更新。Claude Code 会话可以涵盖从调查事件到重构服务，再到分析数月数据的各个方面。
[查看原文](https://claude.com/blog/artifacts-in-claude-code)
